# OEE Verticals

Verticals are pre-configured OEE implementations for specific deployment domains. Each vertical inherits the enforcement primitives from `/core/` and adds domain-specific policy templates, integration examples, and documentation.

Directory names use plain domain words for discoverability. Full product names and acronyms are in each vertical's `README.md`.

## Available Verticals

| Directory | Product Name | Description |
|---|---|---|
| [`venture/`](./venture/) | Open Venture Engine (OVE) | Autonomous venture capital and fund deployment |
| `healthcare/` | Open Healthcare Engine (OHE) | *Planned* |
| `banking/` | Open Banking Engine (OBE) | *Planned* |
| `enterprise/` | Open Enterprise Engine | *Planned* |
